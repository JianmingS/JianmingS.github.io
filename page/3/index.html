leetcode 62. 不同路径

发表于 2021-06-05 更新于 2021-06-06 分类于 leetcode 阅读次数：

解题思路

定义状态：
dp[i][j] := 从 (0, 0) 走到 (i, j) 不同的路径数目

动态规划转移方程：
dp[i][j] = dp[i][j-1] + dp[i-1][j]

状态压缩：
dp[j] += dp[j-1]
